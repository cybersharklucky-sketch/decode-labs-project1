# Python To-Do List Application

A simple console-based To-Do List application developed in Python as part of DecodeLabs Project 1.

## Features

- Add new tasks
- View all tasks
- Mark tasks as completed
- Delete tasks
- Automatic JSON data storage
- Object-Oriented Programming (OOP)
- Clean and modular code
- Input validation

## Technologies Used

- Python 3
- Object-Oriented Programming
- JSON
- File Handling

## Project Structure

.
├── todo.py
├── tasks.json
└── README.md

## How to Run

1. Clone the repository

git clone https://github.com/cybersharklucky-sketch/decode-labs-project1.git

2. Open the project folder

cd decode-labs-project1

3. Run the application

python todo.py
